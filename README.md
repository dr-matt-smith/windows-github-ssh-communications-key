# windows-github-ssh-communications-key

You need to setup, and share an SSH key to allow communication from your Windows comptuer to GitHub.

If you don't have such a key setup, then you'll get this error when you try to clone/push repos between a Windows computer and GitHub:
- ![](/images/1_ssh_key_error.png)

## How to solve the "git@github.com: Permission Denied (publickey)" problem

Those nice people at dev.com have published a step-by-step guide on how to setup and share an SSH key on your Windows computer here:
- https://dev.to/nileshcodehub/how-to-fix-permission-denied-publickey-when-using-git-with-github-on-windows-3c9f

I have created a video of me working through these steps, using PowerShell on my (virtual) Windows computer. It all is pretty straightfoward if you follow the steps in the link above.
